# ControllerCursorTS
A port of MrAsync's Controller Cursor module.

## Installation:
```
npm i @rbxts/controller-cursor
```

## Example Usage
```TS
import { ShowCursor, HideCursor } from "@rbxts/controller-cursor";

ShowCursor();
wait(5);
HideCursor();
```
